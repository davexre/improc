ПАРЛАМЕНТАРНИ ИЗБОРИ

За определяне на вида на секционния протокол, въвеждаме следните специални означения :
    П   - Протокол на Подвижната секционна избирателна комисия
    К   - Протокол на Корабна преброителна комисия
    Д   - Протокол-грама на ДКП
