Download Instructions

  1. Download the compressed zip file download.zip (95 kByte).
  2. unzip download.zip creates a directory VoroGlide.
  3. cd VoroGlide; appletviewer index.html.en starts the program.
The complete set of files including source code is contained in all.zip (147 kByte).
